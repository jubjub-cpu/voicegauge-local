# Release Notes

## v1.0.0 - Local Audio QA Workbench

- Added four deterministic speech-shaped WAV fixtures: clear, clipped, noisy, and pause-heavy.
- Added real browser Web Audio decoding and local audio upload.
- Added peak, RMS, clipping, silence, quiet-floor, and pause metrics.
- Added Canvas waveform rendering with timed clipping, silence, and noise overlays.
- Added visible policy thresholds and a clear-baseline comparison.
- Added a separate human delivery gate with evidence required for accepted exceptions.
- Added local JSON report export and review audit.
- Added logic, fixture, repository, desktop/mobile browser, accessibility, privacy, screenshot, load-failure, HTTP, and deployed-test paths.

No real voice, customer recording, upload, transcription, identity inference, emotion inference, or production handoff is used.
